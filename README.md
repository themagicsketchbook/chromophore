# Chromophore

Chromophore is a Game Engine written in JavaScript for building 2D and 3D desktop applications.

* **2D & 3D Tile Maps** 

* **2D & 3D Fixed Scenes**

* **Canvas and WebGL Support** 

* **Chromium-based JavaScript Rendering** 

* **Release on Windows & Mac with a Common Codebase**
